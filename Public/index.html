API: Timestamp microservice

This microservice converts timestamp from/to unix and natural language and works as follows:
    1-) Enter the web link the explorer address (see below for example entries)

    Example usage:

    https://time-service-stamp-api-mesarikaya.c9users.io/December%2015,%202015
    https://time-service-stamp-api-mesarikaya.c9users.io/1450137600

    Example output:

    { "unix": 1450137600, "natural": "December 15, 2015" }
    2-) If entry is invalid - it does not contain a date or Unix timestamp, it returns null for those properties. in this case check your entry and enter a valid one as given in examples.
